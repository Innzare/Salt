Нужно сверстать одну responsive страницу. PSD для мобильной версии нету, поэтому используйте свое чувство прекрасного.

Требования:
1. Pixel Perfect
2. Responsive
3. Кроссбраузерность. 3 последние версии GC, FF, Safari, Edge + IE11
4. CSS framework по желанию
5. Flex обязательно
6. Все плагины на свой выбор
7. Все подключенные плагины должны работать (pop-up с видео, слайдер и плагин для твиттера)
8. Smoothscroll для блока “Find out more”
9. В выборе hover эффектов также ориентируйтесь на свое чувство прекрасного

К выполненному заданию приложите файл с настройками вашего сборщика проектов.

Оцениваться будут: соответствие страницы указанным требованиям, структура файлов и структура самого CSS.  


Задание сдать ссылками на github репозиторий и на сайт на почту t@purrweb.com. 
В теме письма укажите “Тестовое задание HTML-верстальщик”.
